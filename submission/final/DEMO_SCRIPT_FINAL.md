# BenchPilot — final 105-second demo script

## 0:00–0:10 — The problem

**Visual:** BenchPilot opening screen and the **Load zinc-air demo** action.

**Voiceover:** Independent inventors create real experimental data, but photos, readings, and rough notes rarely become reproducible evidence. BenchPilot turns that messy record into a rigorous next move.

## 0:10–0:25 — Capture

**Visual:** Load the zinc-air replay. Show the captured notes and provenance labels.

**Voiceover:** This real zinc-air prototype measured 1.692 volts fresh open circuit and approximately 1.100 volts under load in an improvised Tic Tac container, using an Analog Devices Scopy voltmeter. BenchPilot preserves those as reported measurements—not conclusions.

## 0:25–0:45 — Structure

**Visual:** Select **Structure**. Move through apparatus, materials, measurements, image observation, unknowns, and safety.

**Voiceover:** GPT-5.6 structured the evidence into a validated record: zinc, potassium hydroxide, the carbon–manganese-dioxide cathode, stainless-steel mesh, measurements, safety, and missing controls. The negative 0.460-volt screenshot minimum remains an uncertain transient, not proven reversal. With no measured current or load resistance, BenchPilot refuses to invent current, resistance, or power.

## 0:45–1:05 — Challenge

**Visual:** Select **Challenge** and show the three hypothesis cards.

**Voiceover:** Instead of agreeing with the inventor, Challenge My Interpretation proposes three falsifiable mechanisms: electrolyte wetting, air-cathode transport, and electrical contact. Each shows supporting evidence, counter-evidence, unknowns, confidence, and the observation that would falsify it.

## 1:05–1:25 — Matrix and test

**Visual:** Show the Hypothesis Matrix, apply the visibly simulated measurement, then select **Test**.

**Voiceover:** The Hypothesis Matrix makes the reasoning visible. Add a clearly simulated matched-replicate outcome and the matrix updates immediately: contact-only loses support, while wetting and cathode construction remain confounded. The next experiment is ranked by information gained per unit effort.

## 1:25–1:40 — Compare

**Visual:** Select **Compare**. Show the chart, causal warning, three-run timeline, and report button.

**Voiceover:** Compare the latest result with the run that collapsed to 0.482 volts and the earlier run that recovered to 1.308 volts. Every chart point comes from validated data. Ten uncontrolled variables prevent attributing the improvement to acrylic binder—or any single change.

## 1:40–1:45 — Close

**Visual:** Open the exportable report, then return to the product mark.

**Voiceover:** BenchPilot turns messy physical experiments into reproducible evidence.
